Práctica 1: Juego de Ajedrez

Objetivos

  • Emplear el método alfa-beta para la implementación de un juego en una máquina

  • Diseñar una función de evaluación que valore la ventaja de una configuración del tablero de juego

El problema

Aplicar el algoritmo alfa-beta al conocido juego de ajedrez. Se os proporciona un entorno gráfico desarrollado en Java que representa el tablero de juego en pantalla y permite la utilización del ratón para mover las fichas, para mover una ficha se pincha sobre ella y seguidamente se pincha en la casilla de destino. Para lanzar el entorno una vez generados los .class,

java Principal

El entorno dispone de cuatro botones:

  • Empezar: visualiza el tablero de juego con las fichas.

  • Cargar: recupera una partida a medias para seguir en el punto en que se quedó.

  • Grabar: guarda la configuración actual del tablero para poder seguir la partida posteriormente.

  • Salir: sale del entorno.

La máquina siempre juega con negras y el usuario con blancas.

Dentro de la clase Tablero hay que insertar el código correspondiente a la fase 1. No es obligatorio implementar la jugada correspondiente al enroque ni la finalización de la partida por tablas.
